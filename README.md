# materia-claim-api
## Usage
Clone the repository:
```shell
> git clone https://github.com/materia-dex/materia-claim-api.git
```
Setup the project:
```shell
> cd materia-claim-api
> npm install
```
Launch the server:
```shell
> node app.js
```
## Swagger
You can test APIs using Swagger UI configured on [localhost:80](http://localhost:80 "localhost:80").
